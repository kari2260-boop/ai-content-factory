## 🔧 如何使用 OpenClaw

### 1. 知识库搜索（自动）

系统在生成内容时会**自动调用** OpenClaw 搜索相关知识：

```
生成内容时
  ↓
自动搜索 OpenClaw 记忆库
  ↓
找到相关的历史观点和案例
  ↓
融入到生成的内容中
```

### 2. 通过 Telegram 控制

你可以通过 Telegram Bot 远程控制：

1. 打开 Telegram
2. 找到 `@xiaok_kari_bot`
3. 发送指令，例如：
   - "搜索 AI教育 相关的笔记"
   - "显示最近的记忆"
   - "帮我采集今日教育热点"

### 3. 通过 Web 控制面板

访问：http://127.0.0.1:18789/

可以：
- 查看所有记忆
- 搜索知识库
- 手动添加记忆
- 查看统计数据

### 4. 知识库注入

通过「📚 知识库管理」页面：
- 收件箱监听：拖文件到 `~/ai-content-factory/data/inbox/`
- URL 抓取：输入 URL 自动抓取内容
- 文本注入：直接粘贴文本

所有注入的内容都会自动写入 OpenClaw 知识库。

## 🎯 OpenClaw 的作用

1. **增强内容质量**：生成内容时引用你的历史观点
2. **知识积累**：所有注入的文档都存入知识库
3. **远程控制**：通过 Telegram 随时随地操作
4. **记忆搜索**：快速找到相关的历史内容
