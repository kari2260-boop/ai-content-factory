## 📝 关于实时选题采集的最终说明

### 🔍 问题分析

经过多次测试，发现以下限制：

1. **知乎热榜**: 403 拦截，无法直接访问
2. **OpenClaw web_search**: 你的 OpenClaw Agent 没有配置 web_search 工具
3. **OpenClaw browser**: 浏览器服务端口冲突
4. **传统 API**: 需要 API key 或被限流

### ✅ 推荐的实用方案

#### 方案 1: 手动采集（最可靠）

每天早上花 5 分钟：

1. 打开浏览器访问：
   - 知乎热榜：https://www.zhihu.com/hot
   - 微博热搜：https://s.weibo.com/top/summary
   - 今日头条：https://www.toutiao.com/

2. 复制 15-20 个教育相关的热点标题

3. 运行脚本生成选题文件：
```bash
cd ~/ai-content-factory
python3 scripts/create_topics.py
```

4. 按提示粘贴标题，系统自动生成 JSON

#### 方案 2: 使用 Claude 生成测试选题（当前）

系统已经准备了 5 个高质量测试选题，可以直接使用。虽然不是实时的，但话题都是长期热点，内容生成质量完全没问题。

#### 方案 3: 配置 OpenClaw web_search（需要配置）

如果你想让 OpenClaw 自动采集，需要：

1. 配置 OpenClaw 的 web_search 工具
2. 或者配置浏览器自动化
3. 参考 OpenClaw 文档：https://docs.openclaw.ai/

### 💡 我的建议

**先用现有的测试选题熟悉系统**，因为：

1. ✅ 内容生成功能完全正常
