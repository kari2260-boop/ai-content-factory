## 🎯 OpenClaw 实时采集说明

由于 OpenClaw 命令行方式比较复杂，我建议你直接通过 **Telegram Bot** 来采集热点：

### 方式 1: 通过 Telegram Bot（推荐）

1. **打开 Telegram**，找到 `@xiaok_kari_bot`

2. **发送采集指令**：
```
请帮我采集今日（2026年2月19日）教育领域的最新热点选题。

要求：
- 访问知乎教育热榜、微博教育热搜、今日头条教育频道
- 采集 15-20 个选题
- 必须是今天或最近3天内的热点
- 与留学规划、AI教育、家庭教育相关

输出 JSON 格式：
[{"title":"标题","excerpt":"摘要","url":"链接","source":"来源","heat_score":"热度"}]
```

3. **等待 Bot 返回结果**（1-2分钟）

4. **复制 JSON 结果**，保存到：
```bash
~/ai-content-factory/data/topics/latest.json
```

### 方式 2: 手动在 Streamlit 中采集

1. 访问 http://localhost:8501
2. 进入「📊 选题采集」
3. 点击「🔄 开始采集选题」
4. 系统会自动尝试多种方式采集

### 方式 3: 使用备用选题（当前）

系统已经准备了 5 个测试选题，你可以先用这些开始：
- AI时代，孩子该学什么才不会被淘汰？
- 留学申请季，如何避开这些常见误区？
- 双减政策下，家长如何重新定位教育目标？
- ChatGPT进校园，老师和家长该如何应对？
- 海外名校录取标准悄然改变，你注意到了吗？

### 为什么选题会重复？

因为当前使用的是备用测试数据。要获取实时热点，需要：
1. 使用 Telegram Bot 手动采集
2. 或配置好网络环境后让系统自动采集

### 建议

**先用现有的 5 个选题测试系统**，熟悉流程后，再通过 Telegram Bot 获取实时热点。

系统的内容生成功能已经完全正常，可以立即使用！
